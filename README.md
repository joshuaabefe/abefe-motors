# Abefe Motors Website 🏎️

A premium, fully responsive car dealership platform for **Abefe Motors**, Nigeria's trusted automobile partner.

## ✨ Features
- **Modern Architecture**: Clean, semantic HTML5 structure.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop viewing.
- **Vehicle Inventory**: High-quality showcase of available automobiles.
- **Booking System**: Integrated test drive booking functionality.
- **Service Hub**: Detailed information on maintenance and customer support.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure and accessibility.
- **CSS3**: Custom styling with a focus on responsive design.
- **JavaScript**: Interactive elements and form handling.
